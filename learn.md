- import pandas as pd -> Imports Panda Library
- pd.read_csv("insert_csv_name.csv") -> Uses Pandas to parse and read the CSV
- A CSV is usually assigned to a variable like df, which stands for dataframe. This is done so that the csv is easier to work with and call methods on, such as df.head().
- df.head() -> Shows first 5 rows by default. Can change the 5 to another value by putting a number between the ().
- df.tail() -> The opposite of df.head()
- df.isna() -> Boolean check for null, none or NaN/NaT values. True if it is null/NaN/Nat/None, False if it is anything else like int, String etc.
- df.sum() -> Adds up the values in a column
- df.isna().sum() -> Sums up the values that are null for each column
- df.isna().sum().sum() -> Shows the overall sum of all column's null/NaN/NaT values
- df["Column_Name"] = df["Column_Name"].fillna("") -> Picks a specific column that has null/NaN/NaT values and replaces it with empty strings.
- df.duplicated() -> Checks for duplicate values per row
- df.duplicated().sum() -> Shows the sum of all the duplicate values per row
- df.describe() -> shows the statistics for each attribute in the dataset.
- numeric_columns_data = df.select_dtypes(include=["number"]) -> Creates a subtable of dataframe with a SELECTED DATATYPE THAT INCLUDES ONLY NUMBERS.
- Correlation Matrix can only be calculated with numbers and can be done between different data types. Hence numeric columns are extracted and then correlation matrix is calculated: numeric_columns_data.corr() . However, running .corr() would work by ignoring non numeric values.
- Writing just df and running would output the whole table.
- To get specific attribute column: df["Column_Name"]
- df["Column_Name"].value_counts() shows the counts of each distinct value in that column
- df["Column_Name"].value_counts().plot(kind="bar") Plots the counts of each distinct value in that column in a graph. Kind can be bar, pie etc.
- import matplotlib.pyplot as plt -> This imports a graph library for labels and more.
- plt.title("Churn (Yes or No)")
plt.ylabel("")
plt.show() -> This just sets a title for the plot. plt is the library called to get the title and ylabel for the graph. plt.show hides the text output above the graph.
- df.groupby("Column_1_Name")["Column_2_Name"].mean() -> Is basically, the numeric average of column 2 values divided by each distinct column 1 value
- df.groupby(["Column_1_Name", "Column_2_Name"])["Column_3_Name"].mean() -> Filters the mean values of Column 3's numeric values by 2 layers of grouping. 
- Churn is bad as customer leaves. More No for Churn is good, but there are situations where Churn is good.
- .plot(kind="bar") -> Takes the functions or calculations before the .plot and creates a graph for them.
- plt.hist(df["Column_Name"]) -> shows the histogram of the spread of the column's numeric values
